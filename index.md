# Welcome to DASH. Data Analysis Starts Here.

<p align="center">ABOUT | <a href="https://github.com/clarelgibson/dash">TEMPLATE</a> | TUTORIAL | <a href="further.md">FURTHER READING</a></p>

DASH is a set of collaborative, modular and reproducible techniques for planning, wrangling, delivering and optimizing data analytics projects. DASH combines several methodologies, including the [Maven Analytics](https://www.mavenanalytics.io) [data analysis workflow](https://mavenanalytics.io/course/thinking-like-an-analyst) and the [Business Event Analysis and Modeling (BEAM)](https://modelstorming.com) approach to dimensional modeling, to produce an end-to-end solution for every data analysis or business intelligence project.

## Who is DASH for?
DASH is intended for business and data analysts or teams who are responsible for planning, delivering and optimizing data analytics projects, from small-scale reports or charts to answer a single question to large-scale data warehouses to support enterprise-level dashboards and KPI monitoring. Think of DASH as a template or framework to ensure all of the elements of a successful project are included in your workflow.

## How to get started?
The first step is to [create a copy of the template repository](https://github.com/clarelgibson/dash/generate). If you're already familiar with the DASH framework, you can start to customize this template right away. If you're new to DASH, this site will provide you with all the resources you need.
